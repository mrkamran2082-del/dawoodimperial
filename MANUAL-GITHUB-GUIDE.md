## üÜò TROUBLESHOOTING

### File Upload Issues:
- **Files too big**: Use smaller images (< 10MB each)
- **Too many files**: Upload in batches (HTML/CSS first, then JS, then images)
- **Upload fails**: Check internet connection, try different browser

### Netlify Issues:
- **Build fails**: Check if _redirects and netlify.toml files are uploaded
- **404 errors**: Verify all files are in repository
- **Admin not working**: Check browser console for errors
